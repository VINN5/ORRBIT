# Orrbit

**Your Intelligent AI Knowledge Assistant** 🤖

A powerful, multi-domain AI assistant that combines **conversational memory**, **long-term user profiling**, and **Retrieval-Augmented Generation (RAG)** to deliver accurate, contextual, and personalized responses across various fields.

---

## ✨ Features

- **Multi-Domain Expertise**: Specialized knowledge in Astronomy, Finance, Technology, Health, Legal, Business, History, Sports, and more.
- **Long-Term Memory**: Remembers user preferences, past conversations, and builds a personal profile over time.
- **RAG-Powered**: Retrieves up-to-date information from academic papers (arXiv) and web sources.
- **Smart Input Guard**: Ensures safe and appropriate interactions.
- **Modern UI**: Built with React + Vite for a smooth user experience.
- **Hybrid Architecture**: Python backend with React frontend.

---

## 🛠️ Tech Stack

### Frontend
- React + React Router
- Vite
- Tailwind CSS (assumed)

### Backend
- Python
- LangChain / LlamaIndex style architecture (inferred)
- RAG with arXiv integration
- Persistent memory system

### Key Modules
- **Memory System** – Conversation + Long-term + User Profile
- **Prompt Engineering** – Domain-specific system prompts
- **Graph-based Reasoning** – Advanced query routing
- **Tool Integration** – Web search, academic fetcher, etc.
